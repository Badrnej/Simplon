# Gestionnaire de tâches

Ce programme est un gestionnaire de tâches simple écrit en C. Il permet à l'utilisateur d'ajouter, modifier, supprimer, filtrer et afficher des tâches, ainsi que de marquer une tâche comme terminée. Les tâches sont stockées en mémoire et sont limitées à un maximum de 100 tâches.

## Fonctionnalités

- Ajouter une tâche avec un titre, une priorité et une date d'échéance.
- Modifier une tâche existante.
- Supprimer une tâche de la liste.
- Filtrer les tâches par priorité ou par date d'échéance.
- Afficher toutes les tâches de la liste.
- Marquer une tâche comme terminée.

## Utilisation

1. Exécutez le programme.
2. Choisissez parmi les options du menu principal en entrant le numéro correspondant.
3. Suivez les instructions pour effectuer l'action désirée.
4. Répétez les étapes 2 et 3 jusqu'à ce que vous choisissiez de quitter le programme.

## Compilation

Pour compiler le programme, exécutez la commande suivante :

gcc main.c function.c -o gestion_taches
